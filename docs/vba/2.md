# if判断

## range

:::warning
会在当前工作表执行
:::

```vb
Sub test()
  Dim i as Integer
  For i = 1 
	Range("a"&i) = i
End Sub
```

## if语句

### 单分支

```vb
if xxx Than
'XXX
end if
```

### 多分支

```vb
if xxx Than
'xxx
	else if
'xxx
end if
```

## 退出for循环

```vb
if xxx Than
	exit for
end if
```
