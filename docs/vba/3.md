# 操作工作表

## 选择表

```vb
'方法1
sheet1.select
'方法2
sheets(1).select
'方法3
sheets("1月").select
```

## 插入表

```vb
'随意插入
sheets.add
'before/after 指定次序，count指定数目
sheets.add after := sheet3,count:=2
'最后一个表后插入
sheets.add after := sheets(Sheets.count)
```

## 重命名

```vb
sheets.name = ""
```

## 删除表

```vb{1}
sheet1.delete

'忽略警告删除表
excel.application.displayAlerts = false

sheet2.delete
```

## 拷贝表

```vb
sheet1.copy after := sheets(sheets.count)
```
