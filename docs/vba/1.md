# for循环

:::tip
书写注释是个良好习惯，按`Tab`控制缩进
:::
## 注释

```vb
‘注释内容
```

## 语法

### 基本语法

```vb
Sub test()
  Dim i as integer
for i = 1 to 50

Next
End Sub
```

### 固定步长

::: tip
删除行会自动跳到原来的下一行

从最后一行开始数，将step改为负数
:::

```vb
sub test(
  dim i as integer
  for i = 1 to 6 step 2
    
  next
end sub
```
