## 更改当前工作目录
使用cd命令更改当前工作目录，参数说明如下
![](/linux/7.png)

绝对路径
绝对路径从根目录开始，直到它的目的地，`pwd`命令显示的路径其实是绝对路径

相对路径
相对路径开始于工作目录，使用特殊符号"`.`"和"`..`（父目录）"

:::tip

可以省略`./`

:::
