## ls
```
ls /usr   //指定目录
ls -l  //显示细节
```
## 确定文件类型
```
file filename
```
## less
less命令是一个用来浏览文本文件的程序
```
less filename
```


*****


| 目录    |  内容   |
| --- | --- |
| /    |   根目录  |
|  /bin   |  包含系统启动和运行所必须的二进制程序   |
|  /boot   |  包含Linux内核、最初的RMA磁盘映像和启动加载程序   |
| /dev    | 包含设备结点的特殊目录    |
| /etc    | 包含所有系统层面的配置文件。它也包含一系列的shell脚本    |
