# shell

>[!tip]
>shell是一个程序，它接受从键盘输入的命令，然后把命令传递到操作系统去执行，几乎所有的Linux发行版都提供一个名为`bash`的shell程序

## 终端仿真器
当使用图形用户界面时，我们需要另一个和shell交互的叫做`终端仿真器`的程序

>[!warning]
>Ubuntu里叫terminal

## 第一次按键
启动终端后，应该看到一行这样的文字
![](/images/linux/1.png)
这叫做shell提示符，通常包括你的用户名@主机名
如果提示符的最后一个字符是“`#`”，而不是“`$`”，那么这个终端会话就有超级用户权限

>[!caution]
>我们建议不要随意使用超级用户权限，除非你知道在干什么
